polyjs
======

This is a polyjs.com's source.

Проект позволяет
	на фронте
		создавать альбомы
		загружать фото
		владельцам удалять и редактировать альбомы и фото
		всем просматривать фото
		всем комментировать фото
	в бекенде
		создавать альбомы
		загружать фото
		удалять и редактировать альбомы и фото
		редактировать комментарии
